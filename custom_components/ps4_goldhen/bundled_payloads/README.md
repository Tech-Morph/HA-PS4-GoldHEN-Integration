# Bundled payloads

If you want payloads to be "auto-installed" into /config/ps4_payloads after HACS install:

1) Put your .bin/.elf files in this folder.
2) The integration will copy them to /config/ps4_payloads on first setup.
3) Existing user files are never overwritten.

Only include payloads you are allowed to redistribute.
